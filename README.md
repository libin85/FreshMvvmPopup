# FreshMvvmPopup
Sample code on how to use Rg.Plugins.Popup with FreshMVVM

This sample code demonstrates how we could set navigation for Rg.Plugins.Popups in Xamarin.Forms Mvvm. The MVVM framework that we use in this example is FreshMVVM.
The code demonstrates how we could navigate from a ContentPage to Popup and a Popup to Popup.
